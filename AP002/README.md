# ROHAND ROH-AP002 MUJOCO PACKAGE

rohand mujoco package

## 1.Prepare

```BASH
pip install mujoco
```

## 2.Controller

| Name            | Description                                             |
| --------------- | ------------------------------------------------------- |
| "index_finger"  | control index finger bending by changing it's position  |
| "middle_finger" | control middle finger bending by changing it's position |
| "ring_finger"   | control ring finger bending by changing it's position   |
| "little_finger" | control little finger bending by changing it's position |
| "thumb_root"    | control thumb rotation by changing it's position        |
| "thumb"         | control thumb bending by changing it's position         |

## 3.RUN

run 'main.py' file and choose type according to the prompt.

```BASH
python main.py
```
